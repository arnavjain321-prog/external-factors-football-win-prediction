02 Modeling: Predicting Home Win Probability from External Factors

In this notebook, we take the modeling-ready dataset created in 01_data_processing.ipynb and:

  1. Load the processed match-level data.
  2. Define the prediction target (home win) and feature set (weather, travel, venue, odds).
  3. Train several models:
    • Baseline (betting odds only, if available)
    • Logistic Regression
    • Random Forest
    • Gradient Boosting
    • CatBoost (gradient boosting on decision trees)
  4. Evaluate models using:
    • ROC AUC
    • Brier score
    • Log loss
    • Accuracy
  5. Compare models and choose a best model.
  6. Save the best model and its test-set predictions for use in 03_reporting.ipynb.

# 5. Baseline: betting odds implied home-win probability (if available)

def evaluate_probs(name, y_true, p_true):
    """Evaluate a vector of win probabilities."""
    auc = roc_auc_score(y_true, p_true)
    brier = brier_score_loss(y_true, p_true)
    ll = log_loss(y_true, p_true)
    acc = accuracy_score(y_true, (p_true >= 0.5).astype(int))
    return {
        "model": name,
        "auc": auc,
        "brier": brier,
        "log_loss": ll,
        "accuracy": acc,
    }

results = []

if "p_home_implied" in df.columns:
    baseline_proba = df.loc[X_test.index, "p_home_implied"].values
    baseline_metrics = evaluate_probs("Baseline  implied odds", y_test.values, baseline_proba)
    results.append(baseline_metrics)
    baseline_metrics
else:
    print("No p_home_implied column found; skipping baseline.")

# 7. Fit models and evaluate on test set

def fit_and_evaluate(name, model, X_train, y_train, X_test, y_test):
    model.fit(X_train, y_train)
    y_proba = model.predict_proba(X_test)[:, 1]
    metrics = evaluate_probs(name, y_test.values, y_proba)
    return model, y_proba, metrics

model_objects = {}
proba_store = {}

for name, model in models.items():
    print(f"Training {name}...")
    fitted_model, y_proba, metrics = fit_and_evaluate(
        name, model, X_train, y_train, X_test, y_test
    )
    model_objects[name] = fitted_model
    proba_store[name] = y_proba
    results.append(metrics)

# Collect metrics into a DataFrame
results_df = pd.DataFrame(results)
results_df.sort_values("auc", ascending=False)

# 8. Confusion matrices at 0.5 threshold

def plot_confusion(cm, title):
    plt.figure(figsize=(4, 4))
    sns.heatmap(cm, annot=True, fmt="d", cmap="Blues", cbar=False)
    plt.xlabel("Predicted")
    plt.ylabel("Actual")
    plt.title(title)
    plt.tight_layout()
    plt.show()

for name, model in model_objects.items():
    y_pred = (proba_store[name] >= 0.5).astype(int)
    cm = confusion_matrix(y_test, y_pred)
    plot_confusion(cm, f"Confusion matrix  {name}")

    • Weather: temperature, humidity, cloud cover, wind, pressure, visibility
    • Venue and crowd variables
    • Travel distance and indicators
    • Betting market implied win probabilities (where available)
  • Trained four models:
    • Logistic Regression
    • Random Forest
    • Gradient Boosting
    • CatBoost
  • Evaluated each model using:
    • ROC AUC
    • Brier score
    • Log loss
    • Accuracy and confusion matrices
  • Compared cross-validated AUC across models.
  • Selected the best-performing model by AUC on the test set.
  • Saved:
    • The trained best model (models/best_model.joblib)
    • Test-set predictions (data/processed/test_predictions_best_model.csv)
